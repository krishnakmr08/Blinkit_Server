A modern React Native app with a Node.js + Socket.io backend for real-time grocery delivery. Track orders live, optimize routes with Google Maps, and enjoy a smooth, interactive UI designed for both customers and delivery partners.

✨ Highlights:

✅ Dual Customer & Delivery Partner roles
✅ Live order tracking with Socket.io
✅ Google Maps integration for navigation & smart routing
✅ Real-time location updates for precise deliveries
✅ Sleek animations for a polished user experience
